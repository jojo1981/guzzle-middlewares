## Installation

### Library

```bash
git clone https://github.com/jojo1981/guzzle-middlewares.git
```

### Composer

[Install PHP Composer](https://getcomposer.org/doc/00-intro.md)

```bash
composer require jojo1981/guzzle-middlewares
```
